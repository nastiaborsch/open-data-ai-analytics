# Open Data AI Analytics

## Назва і мета проєкту
**Назва:** Open Data AI Analytics

**Мета:** виконати базовий цикл аналізу відкритих даних з використанням Git/GitHub та Python: завантаження даних, перевірка якості, дослідження даних і візуалізація результатів.  
У межах роботи також відпрацьовується Git workflow: feature-гілки, merge, pull request, розв’язання merge-конфлікту та версіонування.

## Джерело відкритих даних
Портал відкритих даних України: https://data.gov.ua/ :contentReference[oaicite:3]{index=3}

**Обраний набір даних:** Дані моніторингу радіаційного фону в Україні (неверифіковані)  
**Посилання на набір:** https://data.gov.ua/dataset/80c116f5-9826-4e8e-87f8-ff5d5342da94 :contentReference[oaicite:4]{index=4}

**Основний ресурс для аналізу:** `SaveEcoBotRadiationControlMeasurements.csv.zip` (результати вимірювання радіаційного фону) :contentReference[oaicite:5]{index=5}

## 3 питання / гіпотези для аналізу
1. Чи містять дані пропуски, дублікати або некоректні значення, які можуть впливати на результати аналізу?
2. Чи відрізняються значення радіаційного фону між різними станціями / локаціями / платформами спостереження?
3. Чи можна виявити аномальні значення або часові зміни показників за допомогою базового аналізу та візуалізації?

## План роботи з даними
- Завантажити основний ресурс `SaveEcoBotRadiationControlMeasurements.csv.zip`.
- Розпакувати CSV-файл у локальну папку `data/raw/` (ця папка не комітиться в Git).
- Використати Python-скрипти в `src/` для завантаження, перевірки якості, дослідження та візуалізації.
